You are a senior full-stack AI/ML architect, computer vision researcher, and product engineer.

Your task is to design and generate a COMPLETE, PRODUCTION-READY, INNOVATION-LEVEL web application called:

“AI-Driven Consumer Product Authenticity Detection System”

The system must detect whether a product is REAL or FAKE using AI/ML, without relying on manufacturer APIs, and must support both ADMIN and USER roles.

========================
CORE SYSTEM PHILOSOPHY
========================
The AI must not magically know brands. It must learn from ADMIN-provided verified reference images and general counterfeit patterns, then compare USER-uploaded product images against those learned feature profiles. Decisions must be probability-based and explainable.

ADMIN FUNCTIONALITY
========================
Admin must be able to:
1. Login securely
2. Add a VERIFIED REAL PRODUCT
   - Product name
   - Category
   - Brand name (as text only, not truth)
   - Images from ALL angles:
     front, back, left, right, top, bottom
   - Label close-up
   - Cap/seal image

   - Create a REFERENCE FEATURE PROFILE

4. Manage users (view, block)

   - Total scans

========================
USER FUNCTIONALITY
========================
User must be able to:
1. Signup & Login
2. Select product they want (e.g., Bisleri)
3. Upload product image from shop
4. AI analyzes product
5. Get result:
   - Likely Real
   - Suspicious
   - Likely Fake
6. View EXPLANATION:
   - Image similarity score
   - Text mismatch
   - Deceptive look-alike detection
   - Anomaly score
7. If FAKE detected:
   - Ask user permission to share location
   - Capture location (country/state/city/district only)
   - Store anonymized data
8. View their scan history

========================
AI / ML PIPELINE (VERY IMPORTANT)
========================
Implement the following pipeline:

1. Image Preprocessing:
   - Resize
   - Normalize
   - Denoise
   - Edge enhancement

2. Feature Extraction:
   - Vision Transformer / EfficientNet for deep features
   - Extract shape, layout, texture, color embeddings

3. OCR Analysis:
   - Extract brand text
   - Extract batch/date
   - Grammar & spelling check
   - Detect look-alike names using:
     - Edit distance
     - Phonetic similarity

4. Similarity Matching:
   - Siamese Network compares user image features vs reference profile
   - Output similarity score

5. Anomaly Detection:
   - Autoencoder detects deviation from real product patterns

6. Decision Engine:
   - Weighted scoring:
     Visual similarity (40%)
     Text & name logic (30%)
     Anomaly score (20%)
     Intent mismatch (10%)

7. Explainability:
   - Use Grad-CAM to show suspicious regions
   - Return human-readable reasons
